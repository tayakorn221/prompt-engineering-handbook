# prompt-engineering-handbook

คู่มือ: **Prompt 0 → Hero** — เขียน prompt ระดับโปร สำหรับคนที่ใช้ AI พอเป็นแล้ว อยากเก่งจริงและเอาไปสร้างของจริง

📖 อ่านออนไลน์: **https://tayakorn221.github.io/prompt-engineering-handbook/** (HTML ไฟล์เดียวจบ ไม่มี dependency)

## เกี่ยวกับเล่มนี้

ภาคต่อของ [Prompt ที่ดี](https://tayakorn221.github.io/prompt-handbook/) — ไต่จาก "ใช้เป็น" สู่ "สร้างเป็น" ผ่าน **14 บท 5 ระดับ**:

- **L0 ปูพื้น** — AI คิดยังไง (next-token, context window), จาก 6 ส่วน สู่โครงสร้างเป็นชั้น
- **L1 ใช้ทุกวัน** — few-shot, chain-of-thought, จัดโครง & คุมรูปแบบ output
- **L2 ข้อมูลจริง** — grounding + กันหลอน, prompt chaining, ตั้งบุคลิกถาวร
- **L3 ยกระดับ** — ตรวจตัวเอง + meta-prompt, วัด/เทียบ prompt, ความปลอดภัย (prompt injection)
- **L4 นักสร้าง** — tool use/MCP, ต่อ output เป็นระบบ/อัตโนมัติ, คลัง prompt ของทีม

เน้นเข้าใจกลไก ไม่ใช่ท่องสูตร · ภาษาไทย พร้อมไดอะแกรม 8 รูป · ส่วนหนึ่งของ [Handbooks](https://tayakorn221.github.io/handbooks/) — เขียนเพื่อเข้าใจ ไม่ใช่เพื่อท่องจำ
